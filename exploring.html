Exploratory Data Analysis

This page will do the data analysis on the Boston Beer Companyâ€™s stock price.


Stock Price with SAM company:

The data collects the stock price of SAM company, which is a company that produces beer in Boston. The data I choose is the adjusted close stock price.
This is what the original data looks like.
We can see a big increase from the end of 2019 to 2020. However, by the middle of 2020, the stock price becomes decrease.
From the original graph, we can judge it has the tendency and not stationary. It is seasonal, not cyclic.
The data should be additive because the seasonal variation looks constant, it does not change when the time series value increases.



Lag plots

These are the lag plots of my dataset.
These lag plots show significant correlation of the stock price. The value will not change in a short period.
Otherwise, when we see lag 9, the correlation decreases but is still significant.
This means even as the time periods become longer, the stock price still has linear relationships with the lags.



Decompositions

This is the decomposition of the stock price.
The decomposition shows the tendency of stock price and seasonality.
However, the randomness seems a little strange around 2020. The reason may be there is a huge and significant increase in 2020 so the graph has not much drop down performance.



ACF and PACF

There are the acf and pacf of the original dataset.
The ACF plot shows the high correlation for the lags which we mentioned before.
On the other hand, the plots indicate our data is not stationarity since it has a significant relationship with previous data.



Augmented Dickey-Fuller Test

By using the Augmented Dickey-Fuller Test, we can see the p-value is 0.9778.
It is greater than 0.05 so we can judge that the data is not stationary.

ACF with some transformation

The first graph is the original data, the second graph is the acf of differencing and the third one is acf of log transformation.
The second and third acf plot show the stationary. Which means the log-transformation was very successful.



New Augmented Dickey-Fuller Test

By using the Augmented Dickey-Fuller Test to the log transformation, we can see the p-value is 0.01.
It is smaller than 0.05 so we can judge that the data is stationary.



Moving Average Smoothing

This is the moving Average Smoothing plots, I use different MA smoothing for my data.
The first graph is plotted by MA = 7, which is a weekly time period. We can see the MA graph is overfit the original graph.
The second graph is plotted by MA = 30, which is a monthly time period. We can see the MA graph has a small curve and follows with the original one.
The third graph is plotted by MA = 90, which is a quarterly time period. We can see the MA graph has a curve and also follows with the original one.
For these graphs, I may choose MA with 30 or 90. We can see the tendency of our data without overfitting or underfitting.
